# Supply Chain Quality Assurance Gateway

A showcase Ethereum and Solidity capstone project.

## Overview

This project is a business-to-business decentralized application that tracks shipment delivery approval and releases payment only when quality verification is completed.

## Current Progress

- Smart contract development completed
- Unit tests completed and passing
- Frontend integration in progress
- Sepolia deployment coming next

## Project Structure

- `contracts/` → Solidity smart contracts, tests, deployment modules
- `frontend/` → React frontend for MetaMask and contract interaction

## Planned Features

- Buyer deposits funds
- Supplier creates delivery request
- Warehouse manager approves or rejects shipment
- Payment released on approval
- Pausable smart contract security
- Sepolia deployment
- React + MetaMask frontend
